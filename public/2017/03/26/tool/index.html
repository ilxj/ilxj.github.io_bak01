-fm SPI Flash mode( ‘qio’, ‘qout’, ‘dio’, ‘dout’)
-ff SPI Flash frequency(‘40m’, ‘26m’, ‘20m’, ‘80m’)
-fs SPI Flash size in Mbit(4m’, ‘2m’, ‘8m’, ‘16m’, ‘32m’, ‘16m-c1’, ‘32m-c1’, ‘32m-c2’, ‘64m’, ‘128m’)
0x3fc000 指定烧录地址
